### ðŸ”¸ Syntax


---

### ðŸ”¸ Example 1: Prevent child re-render

```jsx

  const [count, setCount] = useState(0);

  const handleClick = useCallback(() => {

  }, []); // stable function

      <Child onClick={handleClick} />
    </>
  );

```
